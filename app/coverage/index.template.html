Code Coverage for QA Rally

The code for QA Rally is here: http://github.com/craigceleste/qarally

The builds are done here: https://travis-ci.org/craigceleste/qarally

Unit tests for QA Rally are run against PhantomJS as part of the build process. PhantomJS is a webkit browser (related to Chrome and Safari) except it does not have a GUI. The JavaScript engine, rendering engine, DOM, etc. are all controlled via command line. It is not intended to guarantee that the code will work in mainstream browsers. It is intended to catch dumb mechanical and logical JavaScript issues. Regular browsers are not available to test with on the build machine, but we can run the tests manually in most desktop browsers (I run them regularly on Sarari, Chrome and IE11).
